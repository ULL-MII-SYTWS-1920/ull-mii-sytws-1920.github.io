    Promise Chaining

    • Promise Chaining
      • A call to promise.then returns a promise, so that we can call the next .then on it.
      • A handler, used in .then(handler) may create and return a promise. In that case further handlers wait until it settles, and then get its result.
      • Promises Chaining fetch example
      • If a .then (or catch/finally, doesn‚Äôt matter) handler returns a promise, the rest of the chain waits until it settles. When it does, its result (or error) is passed further.
